# Azure Cloud Resume Challenge Project

I recently completed the foundation of the Azure Cloud Resume Challenge as part of my journey from technical support to cloud engineering.

## What I Actually Built:

### Frontend Implementation
- Created a simple HTML/CSS resume website
- Successfully deployed it as a static website using Azure Storage
- Set up a custom domain to make it professionally accessible
- Configured HTTPS through Azure CDN for secure browsing

### Basic CI/CD Implementation
- Created a GitHub repository to store my code
- Set up a straightforward GitHub Actions workflow to automate deployment
- Implemented basic tests to ensure the site functions properly before deployment

This project gave me hands-on experience with fundamental Azure services and helped me understand the basics of CI/CD processes. I'm still working on implementing the visitor counter functionality with Azure Functions, which will add backend capability to my current static implementation.

While modest in scope, this project represents an important first step in applying my cloud knowledge in a practical scenario and demonstrates my ability to deploy and secure a simple cloud-hosted website.
